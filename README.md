# AK Sick Announcement Generator

A simple browser-based tool for Al Khaadem to generate formatted sick du'a announcements for WhatsApp and Telegram.

## Features

- Add one or more sick persons with name, gender, and relation to jemaah
- Gender-aware Arabic du'a with correct pronouns (him / her / both / them)
- One-click copy for WhatsApp and Telegram formats
- Live preview before copying
- No server or installation required

## Usage

1. Open the live site: **https://YOUR-USERNAME.github.io/ak-sick-announcement/**
2. Fill in the sick person's name, gender, and relation
3. Click **+ Add Another Person** if needed
4. Click **Generate Announcement**
5. Click **Copy for WhatsApp** or **Copy for Telegram**
6. Paste into your broadcast channel

## Arabic Du'a Pronouns

| Count | Suffix | Transliteration |
|-------|--------|----------------|
| 1 male | yashfiyahu | ay yashfiyahu |
| 1 female | yashfiyaha | ay yashfiyaha |
| 2 persons | yashfiyahuma | ay yashfiyahuma |
| 3+ male/mixed | yashfiyahum | ay yashfiyahum |
| 3+ all female | yashfiyahunna | ay yashfiyahunna |
